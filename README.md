# ajajani

[![npm version](https://badgen.net/npm/v/ajajani)](https://npm.im/ajajani)

Generate **A**d**j**ective, **A**d**j**ective, **Ani**mal strings

## Install

```bash
npm i ajajani
```

## Usage

### ESModule

```typescript
import { generate } from 'ajajani'

const result = generate()
```

### CommonJS

```typescript
const { generate } = require('ajajani')

const result = generate()
```
